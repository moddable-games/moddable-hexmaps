Overview

Moddable Hexmaps is a shared hex map rendering engine and randomiser framework. It supports three board games out of the box and can be extended with custom game configs.

URL Parameters

Control the generator via query parameters:

ParameterValuesDescription
gamenukes, talisman, twilightWhich game to generate
seedany integerRNG seed for reproducible maps
size2–6 (nukes), 4–5 (talisman), 3 (twilight)Ring count / map size
players2–6 (varies by game/size)Player count for base placement

Embedding

Embed a hex map generator in any page:

<iframe
  src="https://hex.moddable.games/generate/?game=nukes&size=4&seed=42"
  width="800"
  height="600"
  frameborder="0"
></iframe>

Games

Nukes

Nuclear war hex maps with 5 map sizes (2–6 rings). Terrain types: water, forest, mountain, plains, desert, and player HQ bases. Supports 2–6 players with pre-defined base positions per map size.

Talisman: Hexed

Hex adaptation of the Talisman board game. Uses concentric rings: inner (crown), middle, river, outer, and optional dungeon ring. Each ring has its own terrain palette.

Twilight Imperium

Galaxy generator for TI4. Fixed 3-ring layout with Mecatol Rex at centre. Tile types: blue (planet systems), red (anomalies), green (home systems), lanes (hyperlanes), and legends.

Architecture

js/xorshift.js     — Seeded PRNG (XorShift128)
js/hex-math.js     — Axial coordinate math, grid generation
js/hex-renderer.js — Canvas-based hex rendering
js/app.js          — Application controller, game switching
data/              — Game configs and hex coordinate data

Coordinate System

Uses axial coordinates (q, r) with ring-based generation. The centre hex is always (0, 0). Each ring N contains 6×N hexes arranged around the centre.

Supports both pointy-top (Nukes, Talisman) and flat-top (Twilight) orientations.
